# Bingo-Desktop-Java
Projeto básico pessoal de um bingo desktop desenvolvido para meu grupo de amigos em homenagem a integrante Sayuri. Foram utilizadas práticas de conceitos básicos de desenvolvimento de software, como CRUD, lógica de jogo e interface gráfica.

A aplicação permite criar e gerenciar prompts de bingo, alterar o numero de células de uma cartela e aleatorizar a exibição dos prompts.
